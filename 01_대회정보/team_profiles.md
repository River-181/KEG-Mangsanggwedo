## 멤버 1: 이승보

### 학력
- 한남대학교 (4년제) 경영정보/빅데이터응용

### 이번 프로젝트 역할
- 제품 구조 설계
- AI 에이전트/오케스트레이터 설계 및 구현
- backend, data flow, runtime 검증 총괄

### 경력
- CNN을 활용한 피부상태 분석 모델 프로젝트
- 학교시설관리 신고 앱 프로젝트

### 수상
- 청년지원사업 공모전 "장관상"
- 창의설계 경진대회 "나눔과 기술 대표상"
- 유클리드소프트 아이디어 경진대회 "대상"

### 프로젝트 내 기여 포인트
- `한 줄 지시 → case → run → approval → document` 루프 구현 주도
- 회귀 테스트, handoff, evidence 정리 구조 설계

## 멤버 2: 김주용

### 학력

### 이번 프로젝트 역할
- frontend UI/UX 정리
- 인프라/배포/실행 환경 정리
- 심사 시연 흐름과 발표 동선 관리

### 경력
- 삼성드림클래스 중학생 멘토링 멘토
- 한국장학재단 사회리더 대학생 고객중심창업(2025) / 범죄심리(2021) 멘토링 수료 (16개월)

### 수상
- 청년지원사업 공모전 "장관상"
- 창의설계 경진대회 "나눔과 기술 대표상"
- 유클리드소프트 아이디어 경진대회 "대상"
- ABC 멘토링 우수사례 공모전 "대상"
- 생성형 AI 챌린지 수상
- 노코드 X AI 챌린지 수상

### 프로젝트 내 기여 포인트

- demo academy, 제출 동선, 심사 리허설 정리

## 팀 특성 요약

- **두 명 모두 AI-native workflow에 익숙한 빌더**
- **이승보**: 운영 구조, 데이터, backend, AI runtime 설계 강점
- **김주용**: UX, 사용자 맥락, demo narrative, frontend polish 강점
- **공동 수상 및 프로젝트 경험 다수**로 팀 호흡이 검증됨
- **발표/심사 대응은 김주용**, **제품/기술 질의 대응은 이승보** 기준으로 역할을 나눔
