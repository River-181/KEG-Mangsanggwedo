# Bet Memo

> "이번 대회에서 우리가 거는 베팅 1개"
> Shape Up: 불확실성을 인정하되, 하나의 방향에 전체를 건다.

---

## 우리의 베팅

> **"Paperclip의 'company as AI' 개념을 한국 교육 기관에 이식하면,
> 운영자와 교사는 교육 본업에만 집중하고
> AI 에이전트 팀이 운영 잡무를 대신 처리하는
> 제로 휴먼 운영이 가능하다."**

---

## 베팅의 구조

```
기존 솔루션의 천장:
  클래스업·온하이 = 루틴(출결·수납) 자동화만
  LGU+·왓퀴즈 = 생기부 단독
  → 두 솔루션 모두 "단일 도구". 역할 분담 없음.

우리의 차별점:
  "If 원장/교사 is the goal-setter, EduPaperclip is the company."
  → 에이전트들이 조직처럼 움직임 (역할·보고·예산·감사로그)
  → 데이터 자산화: 차량·법무·강사·매출이 맥락 속에서 통합
  → 로컬 LLM 옵션: 원본 데이터 이탈 없이 운용 가능
```

---

## 이 베팅이 옳다면 (성공 조건)

1. 심사위원이 "단순 챗봇이 아니라 AI 팀이다"라고 느낀다
2. 2분 데모에서 에이전트 3개가 병렬 실행되는 장면이 나온다
3. "확장성"에서 다른 팀과 확연히 차별화된다
4. KEG 주최사(IT 학원)가 "우리한테 바로 쓸 수 있겠다"고 반응한다
