| 항목 | 한국 | OECD 평균 | 비고 |
|------|------|-----------|------|
| 주당 총 근무시간 (중등) | **43.1시간** | 41.0시간 | OECD 평균 초과 |
| 주당 행정업무 시간 | **6시간** | 3시간 | **OECD 1위**, 평균의 2배 |

> 출처: [OECD TALIS 2024 - 에듀프레스](https://www.edupress.kr/news/articleView.html?idxno=21271), [한국경제](https://www.hankyung.com/article/202510101535i)

**해석**:
- 교사는 수업 시간보다 행정·민원 처리 시간이 더 많음
- 행정업무가 OECD 1위라는 것은 현장성 근거로 강력하게 사용 가능
- 학부모 민원 스트레스 세계 2위 = 민원 처리가 가장 강한 페인포인트

## 공교육 교사 업무 구성

### 담임교사 주요 업무 항목

| 업무 카테고리 | 세부 항목                    | 비고              |
| ------- | ------------------------ | --------------- |
| 수업 관련   | 수업 준비, 교안 작성, 자료 제작      |                 |
| 생활기록부   | 세부능력 및 특기사항, 행동특성, 출결 기록 | 학생 1명당 1~2시간 추정 |
| 학부모 응대  | 민원 전화/문자, 상담 일지 작성       | 야간·주말 포함        |
| 반 행정    | 출결 관리, 가정통신문, 체험학습 신청    |                 |
| 학교 행정   | 공문 처리, 위원회, 각종 보고서       | 주 6시간 행정의 대부분   |
| 학생 지도   | 생활교육, 상담, 방과후 지도         |                 |

### AI 개입 가능 구간

- **생활기록부**: 이미 LGU+ 슈퍼스쿨, 왓퀴즈가 있음 → 차별화 어려움
- **학부모 민원 응대**: AI 솔루션 미흡 (LGU+ 슈퍼스쿨도 단순 출결 알림 수준)
- **행정 아웃풋 자동화**: 공문·보고서·회의 자료 초안 = 아직 공백

## 사교육 학원 원장 업무 부담

### 핵심 수치

- 원장 하루 행정업무 3시간 = **월 300만원 손실** (bati.ai 분석)
  - 행정 비용: 주 3시간 × 월 = 약 180만원
  - 놓친 신규 상담 기회비용: 월 60만원
  - 재원생 이탈 관리 소홀 손실: 월 60만원
- 중소형 학원: 원장 + 강사 2~3명이 전부
- 운영 실태: 주 7일, 공휴일 없음, 강의+경영+학생관리 병행
- 소형 학원 평균 수명: 5년

### 학원 원장 주요 업무 항목

| 업무 카테고리    | 세부 항목                   | 현재 자동화 여부          |
| ---------- | ----------------------- | ------------------ |
| **출결 관리**  | 등하원 확인, 결석 처리, 학부모 알림   | ✅ 클래스업/온하이가 커버     |
| **수납 관리**  | 학원비 청구, 연체 알림, 결제 확인    | ✅ 클래스업/온하이가 커버     |
| **학부모 알림** | 등하원 알림, 가정통신, 공지        | ✅ 알림톡 자동 발송        |
| **신규 상담**  | 문의 응대, 상담 예약, 등록 안내     | ❌ 사람이 직접 처리        |
| **재원생 관리** | 이탈 징후 파악, 재등록 독려        | ❌ 수작업 or 없음        |
| **강사 관리**  | 스케줄 배정, 성과 파악, 수업 준비 확인 | ❌ 구두·카카오톡          |
| **학부모 민원** | 불만 접수, 응대, 에스컬레이션       | ❌ 원장 직접 처리         |
| **시설·차량**  | 정비 일정, 예약 관리            | ❌ 수작업              |
| **매출 보고**  | 지점별 매출, 재등록률, 수익 분석     | ⚠️ 일부만 (수납 데이터 기반) |

**자동화 공백**: 신규 상담 → 이탈 예측 → 강사 관리 → 민원 처리

## 도메인별 학원 특성

| 학원 유형 | 특화 운영 요소 | 공통 vs 특화 |
|-----------|---------------|-------------|
| 보습/교과 | 시험 기간 집중, 자료 제작 폭증, 강사 배정 복잡 | 공통 |
| IT/코딩 | 프로젝트 기반 수업, 포트폴리오 관리, 취업 연계 | KEG와 직결 |
| 태권도 | 승급 심사, 출석 포인트, 학부모 공연/대회 일정 | 특화 |
| 운전면허 | 차량 예약, 시험 일정 관리, 도로연수 스케줄 | 특화 |
| 예체능 | 발표회·공연 준비, 개인 연습 시간 배정 | 특화 |
| 피트니스 | 운동 일지, PT 예약, 회원권 만료 알림 | 특화 |

**시사점**: 태권도·운전면허·예체능은 도메인 고유 운영 요소가 있어 MVP에서 별도 처리 필요. 보습/IT 학원이 가장 범용적.

## 종합 업무 지도 (사교육 기준)

```
[운영자(원장)]
  신규상담 → 등록 → 반배정 → 수업중 → 재등록/이탈 예측
      ↕             ↕            ↕
  학부모민원    강사스케줄    매출리포트
  (비정형)     (조율·피드백)  (지점별)

[강사]
  수업준비 → 수업진행 → 학생평가 → 학부모보고
               ↕
           보강/일정조율
```

**현재 솔루션 커버**: 등록 이후의 출결·수납·알림
**현재 솔루션 공백**: 신규상담, 학부모 비정형 민원, 강사 관리, 이탈 예측
