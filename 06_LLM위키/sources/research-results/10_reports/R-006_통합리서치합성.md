## 2. 소스별 핵심 발견 요약

### 2.1 Gemini (공식 보고서급 분석 4건)

**새로 확인된 핵심 정보:**

1. **"필코노미(Feelconomy)"** — 2026 사교육 핵심 트렌드. 학부모의 감정과 기분을 선제적으로 설계·관리하는 마케팅. 사교육 원장들의 AI 수용성이 매우 높음.

2. **"AX(AI Transformation) 조직"** — 대형 학원 원장들이 데이터 분석·AI 기반으로 조직 재편 중. "π형 인재" 발굴이 핵심 과제. → EduSwarm 수용 환경 양호.

3. **4세대 나이스 "가짜 노동"** — 업무 경감 목적으로 도입되었으나 실제로는 데이터 중복 입력·오류로 행정 부담 증가. 공교육 교사들의 AI 솔루션 수요 높음.

4. **학령인구 2026년 500만 붕괴** — 초1 신입생 29만으로 급감. 사교육 생존 위기 → 솔루션 수요 증가.

5. **민원대응팀 있어도 79.3% "변화 없음"** — 제도적 장치가 있어도 실질적으로 담임에게 책임이 돌아옴. 시스템 수준의 해결책 필요.

6. **교사 68% 정서적 소진** — OECD 최상위. "교사 21% 후회" 54개국 중 1위.

7. **민원 풍선 효과** — 공교육 민원 방어벽 높아지자 학부모들이 사교육으로 민원을 옮기는 현상.

### 2.2 Grok (X 현장 날것 목소리 — 바텀업 근거)

**30건+ X 포스트 분석 결과:**

**학원 원장 현장 목소리:**
- "원장이 고나리질하고 시키는 일은 많고 돈은 존만하게 준다" → 강사 관리 병목
- "상가 주차장 때문에 인생 피곤" → 차량/시설 관리 현실
- "원장실로 울면서 들어오는 학생" → 민원이 운영자에게 직접 집중

**담임 교사 현장 목소리 (레전드 민원):**
- "수업 중 목소리가 커서 아이가 잠 못 잔다"
- "현장학습 버스에서 아이가 똥 쌌는데 왜 비싼 바지 샀냐"
- "공개 사과하라", "선생님 관상이 고집 세 보인다"
- "사랑을 더 주라", "급식 반찬 잘게 잘라달라"

**Grok X 리서치 총평:**
- 공통 페인 1위: **학부모 민원** (학교·학원 모두)
- 운영자: **강사·차량·매출·시설** 4중고
- 교사/담임: **민원·생기부·반 운영** 3중고
- "번아웃", "힘들다", "AI가 대신 해줬으면" 매일 등장

### 2.3 Perplexity (구조화된 정량 분석 4건)

**새로 확인된 핵심 정보:**

1. **교사 악성 민원 경험 46.8%** (교사노조연맹, 교사 4,068명 설문, 2025년)
2. **민원 경로 84%가 교사 개인 휴대전화** — 공식 창구가 있어도 개인 연락이 압도적
3. **92.1% "민원 일원화 필요"** — 교사 스스로 AI/대응팀 위임 원함
4. **생기부 주당 5~8시간** (배재대 설문, 교사 320명) — 주말·공휴일에 처리 45.6%
5. **노란봉투법 2026.3 시행** — 강사 근로자성 리스크 급증. 원장의 법적 부담 증가.
6. **최저임금 10,320원** — 보조 인력 고용 축소 → 원장이 잡무 직접 처리
7. **사교육비 29.2조, 참여율 80%** (2024년, 교육부·통계청)
8. **학원 선택 시 강사 역량 47%** — 강사가 재등록·이탈의 핵심 변수

## 3. 업무 구조 통합 지도

### 3.1 담임교사 — 주당 시간 배분 (다중 소스 통합)

```
총 근무: 43.1시간 (OECD 기준) / 52시간 (자기보고 기준)

수업:     18.7h  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
행정   6.0~8.0h ▓▓▓▓▓▓▓▓  ← OECD 1위 (평균 3h)
수업준비:  6.8h  ▓▓▓▓▓▓▓
생기부:   5~8h  ▓▓▓▓▓▓▓▓  ← 주말/공휴일 처리 45.6%
학생상담:  3.8h  ▓▓▓▓
과제채점:  3.7h  ▓▓▓▓
학부모민원: 측정 불가 (야간·주말 포함, 개인 폰 84%)
```

**AI 개입 가능 최대 구간**: 행정(6-8h) + 생기부(5-8h) = **주 11~16시간**

### 3.2 학원 원장 — 업무 카테고리별 자동화 현황

| 업무 | 현재 자동화 | 자동화 공백 | AI 필요도 |
|------|:-----------:|:-----------:|:---------:|
| 출결·수납·알림 | ✅ 클래스업/온하이 | — | 낮음 |
| 신규 상담 | ❌ 없음 | 전체 | **높음** |
| 재원생 이탈 예측 | ❌ 없음 | 전체 | **높음** |
| 학부모 민원 | ❌ 없음 | 전체 | **높음** |
| 강사 관리·평가 | ❌ 없음 | 전체 | **높음** |
| 정책·규제 브리핑 | ❌ 없음 | 전체 | **중간** |
| 매출·재등록 분석 | ⚠️ 수동 | 인텔리전트 분석 | **높음** |

**신규 법적 부담 (2026)**: 노란봉투법 → 강사 근로자성 관리 + 계약 아카이빙 필요

## 4. 경쟁 화이트스페이스 최종 확인

### 이미 커버된 영역 (진입 금지)

| 솔루션 | 커버 범위 |
|--------|-----------|
| 클래스업, 온하이, 랠리즈 | 출결·수납·알림 (학원) |
| LGU+ U+슈퍼스쿨 | 생기부 AI 초안 + 출결 (공교육, B2G) |
| 왓퀴즈, AI생기부앱 | 생기부 단독 |
| 하이톡, 민원대응팀 시스템 | 민원 접수 채널 (단, AI 없음) |

### 아무도 안 하는 영역 (EduSwarm 기회)

1. **역할 기반 AI 에이전트 오케스트레이션** — 한 줄 지시 → 3개 업무 동시 처리 (7개 소스 모두 공백 확인)
2. **신규 상담 → 등록 → 이탈 예측 전 과정** 인텔리전트 처리
3. **학부모 비정형 민원** 맥락 파악 + 초안 + 에스컬레이션 판단
4. **강사 성과/피로도 자동 분석** (Perplexity: 강사 역량이 재등록 핵심 변수)
5. **생기부 행동특성 + 민원 초안 통합** (공교육, 시스템이 없는 영역)

### 신규 증거 반영 후 재채점

| 항목         |         P-001 학원 원장 Swarm         | P-002 담임교사 Swarm  |
| ---------- | :-------------------------------: | :---------------: |
| **현장성**    |         ★4 → **★4** (유지)          | ★5 → **★5** (유지)  |
| **AI 적합성** |        ★5 (노란봉투법으로 복잡도 증가)        | ★4 (92.1% 위임 원함)  |
| **7일 MVP** | ★3 → **★4** (AX 조직 트렌드으로 수용성 높아짐) |      ★4 (유지)      |
| **데모 전달력** |          ★4 (필코노미 맥락 추가)          | ★5 (레전드 민원 사례 추가) |
