# 한국 학원 원장과 교사들이 매일 반복적으로 처리하는 행정 업무와

불편한 점을 찾아줘. 특히 학부모 소통, 출결 관리, 수납 알림,
시험 일정 관리, 반 편성, 숙제 피드백, 수능/모의고사 일정 연동 등
실제 현장에서 자동화되면 좋겠다고 느끼는 루틴 업무 위주로.
커뮤니티(에듀인뉴스, 학원가 카페, 블라인드 학원 관련 글) 기반으로
실제 목소리를 반영해서 정리해줘.

GitHub과 MCP 생태계에서 한국 서비스와 연동 가능한 MCP 서버,
CLI 도구, Agent Skill을 찾아줘.
찾아야 할 범위: 카카오톡/카카오알림톡, 네이버 예약/스마트플레이스,
국내 SMS 발송(솔라피, 알리고 등), 수능/교육과정평가원 일정 API,
한국 법령 검색(korean-law-mcp), 국내 결제(토스페이먼츠, KG이니시스),
학교/학원 관련 공공데이터 API.
GitHub 레포 링크, 설치 방법, 현재 상태(활성/비활성)도 함께 알려줘.

학원 교사와 직원들의 일상 편의를 높여주는 AI 스킬 아이디어를 찾아줘.
예시: 학원 주변 맛집 추천(네이버 지도/카카오맵 연동),
점심 메뉴 랜덤 추천, 날씨 기반 옷차림 알림,
근처 편의점/카페 영업시간 확인, 교사 생일 챙기기,
학원 근처 주차 정보.
비슷한 컨셉으로 이미 구현된 MCP나 Claude Skill이 있는지도 같이 찾아줘.
k-skill(github.com/NomaDamas/k-skill) 레포에 있는 스킬들도 참고해서
학원에 적용 가능한 것들을 골라줘.

한국 학원 원장·교사들이 매일 반복하는 행정 루틴은 출결·수납·카톡 공지·성적 리포트·반 편성·상담 기록 등 “카카오/문자+엑셀+관리프로그램”을 왔다갔다하며 처리하는 일들이고, 이를 줄이려는 SaaS·MCP 서버들이 이미 꽤 나와 있습니다.  아래에 현장 페인 포인트 → 한국 서비스용 MCP/CLI/스킬 → 학원용 편의 AI 스킬 아이디어 순서로 정리했습니다.[^1][^2][^3]

## 1. 학원 현장의 반복 행정 업무·페인 포인트

### 학부모 소통·카카오톡·알림장

- 많은 학원들이 여전히 **수업 끝날 때마다 개인 휴대폰으로 카톡·문자를 보내며 숙제·공지·상담 내용을 공유**합니다.[^3]
    - 어나더클래스(학원 관리 솔루션)는 “밤에도 ‘선생님 내일 숙제가 뭔가요?’ 같은 문의 카톡이 계속 울린다”는 교사 피로감을 줄이기 위해, 연락처 비공개 채팅(상담톡)·반별게시판·개인 알림장 기능을 제공한다고 설명합니다.[^3]
- 당근마켓 학원 원장 게시글에서는 “원장·선생님들이 제일 귀찮아하는 것들만 모아서 카톡 자동 발송되게 만들었다”고 하면서,
    - 수업 리포트, 공지사항, 학업 성과 리포트(성적표), 개별 시간표를 **매번 직접 타이핑해서 보내던 일을 자동화**했더니 “수업 끝나고 폰 붙잡고 있는 시간이 싹 사라졌다”고 말합니다.[^4]

→ 학부모 소통의 핵심 페인 포인트는 **(1) 반복 입력, (2) 개인 전화번호 노출, (3) 업무·비업무 카톡이 24시간 섞여 들어오는 것**입니다.[^4][^3]

***

### 출결 관리·등하원 알림

- 출결버스(출결 관리 프로그램) 도입 전에는
    - 선생님이 **종이에 출석 체크 → 수업 후 학부모에게 일일이 문자 발송**을 했고, 놓치는 경우 학부모에게 전화가 오는 일이 많았다고 합니다.[^2]
- 도입 후에는
    - 학생이 등원 시 **출결번호만 입력하면 시스템이 출결을 기록하고, 학부모에게 등·하원 SMS/알림톡을 자동 발송**합니다.[^2]
    - 교직원은 출결 관리에 거의 시간을 쓰지 않고 수업에 집중할 수 있게 되었다고 후기를 남깁니다.[^2]
- 다른 블로그들도 학원 출결 관리를 위해 **카카오 알림톡+SMS를 묶은 솔루션(365매니저 등)**을 소개하며, 학생이 번호를 입력하면 자동으로 알림톡/문자가 나가는 구조를 권장합니다.[^5][^6]
- 유아/학원용 소통앱(키즈노트·클래스노트)은 개인정보 처리방침에서 **전자출결(출결상태, 등·하원 시간), 출석부, 청구서 발행, 알림장·공지사항** 같은 항목을 별도로 다루고 있어, 출결+알림이 핵심 기능임을 보여줍니다.[^7]

→ 출결·등하원 관련 루틴은 **수기 출석부 기록 → 등·하원 안내 문자/카톡 발송 → 출결 이력 정리**의 반복이며, 이를 자동화한 프로그램이 “행정 업무의 핵심 엔진”이라는 표현까지 쓰입니다.[^8][^2]

***

### 수납·미납 관리·수납 알림

- 출결버스 후기에서는 **매월 청구서 제작, 납부 기한 안내, 미납자 전화·문자 연락**이 원장에게 가장 큰 스트레스였다고 말합니다.[^2]
    - 도입 후에는 **결제일에 맞춰 모바일 청구서 자동 발송, 비대면 카드·계좌이체·카카오페이 결제, 수납 현황·월별 매출 통계 자동 집계** 기능으로 수납 업무가 크게 줄었다고 합니다.[^2]
- ‘온길 OMS’는 여러 학원 데이터를 분석한 결과,
    - **수납 관리, 성적 발송, 출결/상담** 세 가지가 전체 행정 시간의 60%를 차지했고, SaaS 도입으로 행정 시간이 12시간/주 → 2.5시간/주(약 79% 감소)까지 줄었다고 밝힙니다.[^1]
- 키즈노트·클래스노트도 청구서 발행·스마트 주문·스토어를 통해 **학원비 청구·정산·배송 알림**을 처리하고, 결제 대행은 토스페이먼츠·올댓페이 등 국내 PG와 연동합니다.[^7]

→ 현장 체감상 **“수납 관련 루틴(청구서 발행, 미납 추적, 알림 발송)”이 행정 업무의 가장 큰 블록**이고, 자동화 솔루션이 가장 직접적인 시간 절감 효과를 주는 영역입니다.[^8][^1][^2]

***

### 시험 일정·성적 리포트·숙제 피드백

- 온길 OMS 사례에서는 **성적 발송**을 별도 업무로 분리해,
    - 기존에는 시험 후 점수 입력→ 성적표 엑셀·한글 작성→ 학부모에게 개별 전송까지 5시간/주(시험 기간 기준) 걸리던 일을
    - “점수만 입력하면 상세 분석 리포트가 1초 만에 생성되고, 링크 하나로 학부모에게 전송”하는 구조로 바꾸어 1시간/주 수준으로 줄였다고 설명합니다.[^1]
- 학원들은 각종 자격·고시·모의고사 일정이 나올 때마다
    - 공지문, 응시 유의사항, 모의고사 일정표를 직접 작성해 홈페이지·블로그에 공지하고,[^9][^10]
    - 수능·6·9월 모의평가는 한국교육과정평가원 공지(PDF·HWP)를 보고 학원 내부 일정표와 연동합니다.[^11][^12]
- 어나더클래스는 상담톡 화면에서 **해당 학생의 수강 정보·학원비·숙제·상담 관리 등을 한 번에 수정**할 수 있게 해서, 숙제 안내·피드백과 상담 기록을 각각 다른 화면·프로그램에서 처리해야 하는 번거로움을 줄이려고 합니다.[^3]

→ 시험·숙제 쪽 루틴은 **(1) 외부 일정(수능·모의고사·모의시험) 긁어와서 학원 캘린더로 옮기기, (2) 성적 입력·리포트 생성·전송, (3) 숙제 안내·피드백 기록**이며, 대부분 “엑셀+한글+카톡” 삼중 작업이라 자동화 니즈가 큽니다.[^11][^1][^3]

***

### 반 편성·시간표·상담 로그

- 학원 관리 프로그램 비교 글에서는 “알림톡 자동 발송은 단순 자동화지만, **‘신규생 문의 → 상담 예약 → 등록 → 수납 → 반 배정’ 전체 프로세스를 자동화**하는지가 진짜 차별점”이라고 지적합니다.[^13]
- 어나더클래스는
    - 반별게시판(반 구성에 따라 공지 노출 제어), 재원생/전체 공지 구분,
    - 학생 선택 후 **동일 화면에서 기본정보·학원비·숙제·상담관리 수정** 기능을 제공해 “상담하면서 곧바로 반·시간·수납 정보를 손보고 저장”할 수 있게 설계돼 있습니다.[^3]

→ 실제 현장에서는 **상담 기록을 보고 적절한 반을 배정하고, 시간표 변경 시 학부모 공지·출결 시스템·수납 시스템까지 모두 일치시키는 작업**이 반복되는 루틴입니다.[^13][^3]

***

### 요약: 자동화되면 좋은 루틴 업무

커뮤니티·후기·솔루션 글을 합치면, 학원 원장·교사가 “자동화되면 좋겠다”고 느끼는 루틴은 대략 다음 축으로 정리됩니다.[^4][^1][^3][^2]

- 학부모 소통
    - 수업 리포트·숙제 안내·공지사항·시험/일정 공지를 카카오톡/알림톡으로 자동 발송
    - 1:1 문의 응대는 개인 번호 대신 상담용 채널(앱·웹)에서 처리
- 출결 관리
    - 학생이 키패드/태블릿에 번호만 눌러 출결 체크 → 학부모에게 즉시 등·하원 알림톡/SMS 발송
- 수납·미납
    - 매월 자동 청구서 발행, 비대면 결제(카드·계좌·간편결제), 미납 자동 리마인드
- 시험·성적·숙제
    - 수능/모의고사·학원 자체 모의고사 일정 자동 가져오기, 학생별 시험·숙제 관리와 성적 리포트 자동 생성·발송
- 반 편성·시간표
    - 상담 기록 기반 자동 반 추천, 시간표 변경 시 출결·수납·공지와 연동하여 한 번에 갱신

***

## 2. 한국 서비스 연동용 MCP 서버·CLI·스킬

요청하신 범위를 중심으로, 현재 공개된 MCP 서버·도구를 정리하면 아래와 같습니다.

### 2.1 카카오톡 / 카카오 알림톡

**1) Kakao Developers API MCP 서버 (카카오톡 메시지)**

- `inspirit941/kakao-bot-mcp-server` – GitHub MCP 서버
    - 카카오 디벨로퍼스 API를 이용해 **카카오톡 ‘나에게 보내기’ 텍스트/피드 메시지**를 보내는 MCP 서버 예제입니다.[^14][^15][^16]
    - 주요 도구
        - `send_text_template_to_me`: 텍스트 템플릿 메시지를 본인에게 발송 (웹/모바일 링크, 버튼 타이틀 등 지정).[^15][^16]
        - `send_feed_template_to_me`: 이미지·설명 등이 포함된 피드 템플릿을 본인에게 발송.[^16]
    - 설치/실행(예시)
        - 로컬 클론 후 `fastmcp` 또는 Node 기반 stdio 서버로 실행, Claude Desktop 등에서 `command: "python"` 또는 `node`로 등록.[^15][^16]
    - 상태
        - 2025년 4월경까지 커밋된 공개 레포로, 개인이 만든 PoC 성격이며 공식 카카오 제공은 아닙니다.[^16][^15]
- `jeong-sik/kakao-api-mcp-server` – GitHub
    - 카카오 API 전반을 감싸는 MCP 서버 예제 레포지토리로, 아직 README·도구 정의가 간단한 수준입니다.[^17]

> 알림톡(카카오 비즈메시지)은 직접 카카오가 아닌 **솔라피·알리고 같은 중계사의 API**를 통해 보내는 경우가 많아서, 아래 SMS·비즈메시지 MCP와 조합하는 쪽이 실무적으로 자연합니다.[^18][^19]

***

### 2.2 네이버 지도·스마트플레이스·예약 근접 기능

**1) Naver OpenAPI MCP 서버 (`py-mcp-naver`)**

- `pfldy2850/py-mcp-naver` – GitHub MCP 서버 (공식 MCP 사이트에 등록)[^20][^21]
    - 네이버 OpenAPI(블로그, 뉴스, 책, 이미지, 쇼핑, **지역(local) 검색 등**)를 MCP 툴로 제공.[^20]
    - 설치
        - `uv sync --dev --all-extras`로 의존성 설치 후, 예:

```bash
fastmcp install src/server.py \
  -e NAVER_APP_ID_EXAMPLE=... \
  -e NAVER_APP_CREDENTIAL_EXAMPLE=...
```

형태로 서버 등록.[^20]
    - 학원 활용
        - `local search` 도구로 “{학원 주소 기준} 500m 이내 분식집/카페/주차장” 같은 검색을 할 수 있어, **맛집 추천·주차장 정보·근처 카페 찾기** 스킬의 기반으로 사용 가능.[^22][^20]

**2) 네이버 길찾기 MCP 서버 (`naver-directions-mcp`)**

- LobeHub MCP 목록에 “네이버 길찾기 MCP 서버 (네이버 지도 API 연동)”가 등록돼 있으며,
    - **길찾기, 장소 검색, 지오코딩**을 MCP 도구로 제공합니다.[^23]
    - 예시 설명에서 `"query": "강남역 맛집"`처럼 키워드 검색을 지원해, 학원 주변 맛집·카페 찾기에도 활용 가능합니다.[^23]

**3) Naver Map Scraper / Place Search (Apify + MCP)**

> 네이버 예약/스마트플레이스 “예약 관리”까지 직접 제어하는 MCP 서버는 아직 공개된 것을 찾기 어려웠고, 현재는 **장소·영업시간 조회 정도까지 지원하는 MCP 서버/Actor**가 주류입니다.[^24][^20]

## 3. 학원 교사·직원 편의를 위한 AI 스킬 아이디어 \& 관련 MCP/스킬

### 3.1 주변 맛집 추천·점심 메뉴 랜덤 추천

- **Naver OpenAPI MCP (`py-mcp-naver`)의 지역 검색 + Naver Map Scraper 조합**
    - Naver local search·지도 스크래퍼는 `"강남 맛집"`, `"학원 주소 인근 카페"` 같은 키워드로 **장소·카테고리·평점·영업시간**을 가져올 수 있습니다.[^25][^24][^20]
- **학원용 스킬 아이디어**
    - 매일 11시 30분 자동 실행:

1. 학원 위치 기준 반경 500m 음식점 목록 조회(Naver local).
2. 오늘 영업 중이고, 최근에 안 갔던 집만 필터.
3. 후보 중 1곳 랜덤 선택 후 “오늘의 점심 추천”을 직원 단톡방(카카오 MCP/알리고 MCP)을 통해 전송.

    - k‑skill 레포는 “SRT, KTX, KBO, 카카오톡, 미세먼지, 우편번호, 블루리본 등 한국인을 위한 생활형 스킬 모음집”을 표방합니다.[^46][^47]
    - 이 중 **블루리본·카카오톡 관련 스킬**이 있다면,
        - 블루리본 추천 맛집 → 학원 인근 지역으로 제한,
        - 카카오톡 스킬 → 내부 직원용 맛집 추천 알림 채널에 재사용
하는 식으로 조합할 수 있습니다.

### 3.2 날씨·미세먼지 기반 옷차림·우산 알림

    - `jezweb/weather-mcp-server`, `CodeByWaqas/weather-mcp-server` 등은 OpenWeatherMap API로 현재 날씨·5일 예보·공기질(AQI)까지 조회하는 MCP 서버입니다.[^48][^49]

- **학원용 스킬 아이디어**

        - “오늘 등원 복장 가이드(코트/우산/장화 등)”를 학부모용 채널에 자동 발송.

    - k-skill에 포함된 “미세먼지” 스킬은 국내 미세먼지 정보를 조회하는 용도로 설계돼 있어,
        - Weather MCP의 기온·강수 데이터와 합쳐 **“오늘은 미세먼지가 심하니 마스크 권장”** 같은 메시지를 구성하는 데 활용 가능합니다.[^47][^46]

가 “일정+날씨+리마인더+메시지” 패턴을 이미 잘 보여주고 있습니다.
- 여기에 한국 특화 스킬 모음인 k‑skill(카카오톡·미세먼지·한국지도·생활 편의)을 섞으면,
    - **글로벌 MCP 서버(날씨·캘린더·리마인더) + 한국 서비스용 MCP/k‑skill(카카오·네이버·결제)** 조합으로
    - 학원 현장의 반복 행정·편의 업무를 꽤 높은 수준까지 자동화할 수 있는 구성이 나옵니다.[^33][^47][^46][^36]

***

원하시면, 위 MCP 서버들을 기준으로 **“학원용 MCP 설계서”**(예: 출결·수납·카톡·시험 일정·맛집·날씨를 하나의 에이전트 워크플로로 묶는 구조)를 그려 드리고, k‑skill 안에서 실제로 어떤 스킬 조합을 쓰는 것이 좋은지까지 더 구체적으로 짜 드릴 수 있습니다.
