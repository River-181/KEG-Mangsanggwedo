# 도구별 리서치 프롬프트

## 목적

도구별 강점을 분리해서 같은 질문을 다른 형태의 증거로 수집한다.

## Gemini Deep Research

```text
2025~2026년 한국 공교육과 사교육의 운영 시스템을 비교 분석해 주세요.
비교 단위는 운영자(교장/원장/행정)와 실무자(담임/교사/강사)입니다.
다음을 표와 요약으로 정리하세요.
1) 하루 업무 흐름
2) 반복 민원과 병목
3) 행정/기록 업무 비중
4) 실제 현장 사례
5) AI agent team이 해결 가능한 문제와 불가능한 문제

반드시 교육부, KEDI, KERIS, OECD TALIS 등 신뢰 가능한 자료를 우선 인용하고,
2025년 이후 사례를 우선 사용하세요.
마지막에는 7일 MVP로 가능한 문제 5개만 남겨 주세요.
```

## Perplexity

```text
2025~2026년 한국 학원 원장, 학원 강사, 공교육 담임교사의 실제 업무 부담과 병목을 최신 자료로 찾아주세요.
우선순위는 민원, 스케줄/운영, 학생 관리, 기록/보고, 차량/시설 관리입니다.
각 항목마다 수치, 기사/보고서 출처, 실제 사례를 붙여 주세요.
그리고 EduSwarm 같은 역할 기반 AI agent 구조가 왜 필요한지 또는 왜 과한지까지 판단해 주세요.
```

## NotebookLM

```text
업로드한 자료를 바탕으로 다음을 합성해 주세요.
1) 사교육 운영자용 swarm이 해결할 수 있는 문제
2) 공교육 담임/교사용 swarm이 해결할 수 있는 문제
3) 둘 사이의 공통 문제와 분기 문제
4) 데모에 적합한 장면 3개

결과는 문제명 / 사용자 / 현재 방식 / 병목 / AI 개입 방식 / 데모 장면 형식으로 정리해 주세요.
```

## 사용 규칙

- 같은 주장이라도 `현장 발화`, `통계`, `기사`, `공식 자료`를 분리 저장한다.
- 스크린샷만 있는 경우 반드시 한 줄 요약과 해석을 같이 남긴다.
- 과장된 바이럴 사례는 `참고`로만 보관하고 핵심 근거로 쓰지 않는다.
