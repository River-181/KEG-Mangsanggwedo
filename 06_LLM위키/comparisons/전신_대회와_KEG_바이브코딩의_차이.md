# 전신 대회와 KEG 바이브코딩의 차이

## Comparison Snapshot

| 축 | 2023~2025 전신 대회 | 2026 KEG 바이브코딩 |
|---|---|---|
| 평가 중심 | 해커톤 MVP 또는 코딩 테스트 | 교육 서비스 MVP + AI 활용 |
| 결과물 | 코드/순위/프로토타입 | GitHub + 배포 + 설명 가능한 AI workflow |
| 사용자 맥락 | 일반 개발 역량 | 교육 현장 문제 해결 |
| 강한 인상 | 코딩 실력, 속도, 프로젝트 역량 | 실무 적합성, 교육 이해, AI orchestration |

## Interpretation

- KEG는 코딩을 잘하는 팀보다 "교육 도메인을 읽고 AI를 활용해 끝까지 제품화하는 팀"을 더 높게 평가할 가능성이 크다.
- 따라서 전신 대회식 접근을 그대로 가져오면 범용 개발 데모에 머물 위험이 있다.
