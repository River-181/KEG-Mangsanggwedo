# 현장 발화 샘플 뱅크

## Summary First

현장 발화는 문제를 추상 명사로 설명하지 않고, 실제로 어떤 순간에 누가 무너지는지를 보여준다. 이 페이지는 문제정의, 데모 시나리오, 제출 문서 문장에 재사용할 수 있는 장면 중심 발화를 모아 둔 reference bank다.

## Role-based Samples

### 운영자 / 원장

- "강사 관리, 차량, 민원, 매출이 한 번에 몰린다."
- "수업 끝나고도 공지와 상담 때문에 폰을 놓지 못한다."
- "주차, 차량, 예약 같은 운영 잡무가 교육보다 더 많이 느껴진다."

### 담임 / 공교육 교사

- "교장이 민원을 교사에게 그대로 전달한다."
- "수업보다 문서와 민원 때문에 체력이 먼저 바닥난다."
- "반 운영 루틴이 민원 한 번에 전부 깨진다."

### 학원 강사

- "밤 10시 수업 끝나면 사실상 하루가 끝난다."
- "민원과 학생 관리를 같이 안고 가는데 페이와 권한은 약하다."
- "휴가 중에도 메시지와 보강 대응이 이어진다."

## Reusable Sentence Patterns

- "반복 문의와 약속 누락이 신뢰 저하로 이어진다."
- "예외 케이스가 쌓이면서 운영자가 기억과 메신저에 의존한다."
- "교육 본업보다 운영 루틴과 감정 노동이 더 크다."

## How To Use

- problem note에는 1개만 넣고, 통계/정책 근거와 붙여 사용한다.
- demo script에는 역할별 장면을 1줄씩만 넣어 과장 없이 쓴다.
- 제출 문서에는 직접 인용보다 의역 문장으로 정리하는 편이 안전하다.

## Caution

- 단일 플랫폼의 발화를 전체 현장으로 일반화하지 않는다.
- 익명 커뮤니티 발화는 항상 다른 source와 교차 확인한다.
