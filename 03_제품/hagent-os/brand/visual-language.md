# Visual Language (시각 언어)

> Paperclip.ing에서 읽은 것: 에디토리얼 스크롤, 크림색 베이스, 대형 세리프 헤드라인, 미니멀한 일러스트레이션, 밝고 가벼운 흰 배경. HagentOS는 이 구조를 한국 학원 운영 맥락으로 치환한다.

---

## 로고 컨셉

### 방향
**아이콘 + 워드마크** 조합 (Paperclip과 동일 구조)

### 아이콘 아이디어 (우선순위 순)

1. **H-링크 아이콘** *(권장)*
   - 굵은 `H` 레터폼 안에 에이전트 연결 노드 점 3개(삼각 배치)가 들어있음
   - 의미: 학원(H)이 에이전트 네트워크로 연결됨
   - 색상: teal #0ea5b0 단색, 모노크롬 대응 가능

2. **지붕+AI 아이콘**
   - 학원 건물 지붕 실루엣(단순화된 삼각형) 아래 teal 색 도트 클러스터
   - 의미: 학원 위에서 AI가 지원하는 구조

3. **승인 체크 아이콘**
   - 원 안의 체크마크 + 미세한 에이전트 pulse ring
   - 의미: HagentOS = 승인으로 완성되는 루프

### 워드마크
- 폰트 방향: Pretendard ExtraBold (한글 대응 최적) 또는 Inter 700
- `Hagent` teal #0ea5b0 + `OS` dark #191f28 로 컬러 분리
- 영문 소문자 혼용: `HagentOS` (브랜드 규칙, 대문자 강제 금지)

## 컬러 팔레트

| 역할 | 색상 | Hex | 사용처 |
|------|------|-----|--------|
| Primary | Teal | `#0ea5b0` | CTA 버튼, 링크, 아이콘 활성, 로고 |
| Dark | Ink | `#191f28` | 바탕 텍스트, 헤더, 다크모드 배경 |
| Background | Off-white | `#f7f6f1` | 랜딩 페이지 베이스 (Paperclip과 동일한 크림 계열) |
| Surface | White | `#ffffff` | 카드, 모달, 입력 필드 |
| Success | Green | `#03b26c` | 완료, 승인, 에이전트 정상 실행 |
| Warning | Amber | `#ffc342` | 주의 필요, 검토 대기 |
| Danger | Red | `#f04452` | 긴급 민원, 이탈 위험 HIGH, 오류 |
| Teal light | Teal 10 | `#e6f7f8` | 배지 배경, hover 상태, 아이콘 배경 |

### 컬러 스토리
- Teal(청록)은 신뢰(파랑)와 성장(초록)의 중간 — 교육 플랫폼의 "혁신적이지만 안전한" 느낌
- Off-white 배경은 학원 문서(A4 용지)의 따뜻함을 디지털로 번역
- Dark ink는 전문성과 가독성, 결코 순수 #000 블랙을 쓰지 않음

## 타이포그래피

### 원칙
한국어 학원 원장 대상 → 한글 가독성 최우선. 모바일(390px) 기준 설계.

| 레벨 | 폰트 | 크기(모바일) | 크기(데스크탑) | 용도 |
|------|------|------------|--------------|------|
| Display | Pretendard Black | 36px | 72px | 히어로 헤드라인 |
| H1 | Pretendard Bold | 28px | 48px | 섹션 타이틀 |
| H2 | Pretendard SemiBold | 22px | 32px | 기능 이름 |
| Body | Pretendard Regular | 16px | 18px | 본문 |
| Caption | Pretendard Regular | 13px | 14px | 메타 정보, 라벨 |
| Code | JetBrains Mono | 13px | 14px | 코드 스니펫 (오픈소스 섹션) |

> 영문 병기 시 Inter 패밀리를 혼용 가능. Pretendard는 Inter 기반이라 시각적으로 자연스럽게 조합됨.

## 일러스트레이션 스타일

### 스타일 정의
- **플랫 + 소프트 그라데이션**: 딱딱한 기하도형이 아니라, 모서리 부드럽고 그림자 없는 현대적 플랫
- **색상 절제**: teal 계열 + warm off-white + 한 가지 포인트 컬러(amber or green)
- **선 두께**: 스트로크 중심보다 면 중심 일러스트
- **캐릭터**: AI 에이전트를 작은 로봇으로 표현하지 않음. 대신 **역할 배지를 단 사람 실루엣** (민원 에이전트 = 헤드셋 아이콘, 스케줄 에이전트 = 캘린더 아이콘) — "AI 팀원" 느낌
- **맥락**: 추상적 데이터 시각화보다 학원 현장 물건(교실 창문, 화이트보드, 폰 알림) 포함

### 금지 사항
- 로봇·기계 이미지 (차갑고 무섭게 보임)
- 파란 머리의 AI 캐릭터 (글로벌 AI 일러스트 클리셰)
- 지나치게 추상적인 회로 배경

## 히어로 이미지 컨셉

> Paperclip의 히어로: 파스텔 하늘 위를 날아다니는 분홍 물고기들 — "자유롭게 운영되는 AI 에이전트들"의 은유. HagentOS는 이를 한국 학원 맥락으로 치환한다.

### 추천 컨셉 A — "원장의 아침" *(1순위)*
- 원장 한 명이 아침에 커피를 들고 폰을 보는 장면 (측면 실루엣, 디테일 절제)
- 폰 화면에는 승인 카드 3개가 떠있음
- 배경: 학원 창문 + 아침 햇살 (teal + amber 그라데이션)
- 의미: "AI 팀이 밤새 일하고, 원장은 아침에 확인만 한다"

### 추천 컨셉 B — "에이전트 팀 오버뷰"
- Isometric 뷰의 학원 건물 단면
- 각 층에 에이전트들이 배치: 1층(민원처리실), 2층(스케줄룸), 3층(이탈감지실)
- teal 컬러의 데이터 흐름선이 건물을 연결
- 의미: "학원을 운영하는 보이지 않는 팀"

### 추천 컨셉 C — "승인 한 번으로" *(모바일 버전)*
- 스마트폰 화면 클로즈업
- 승인 버튼 하나 위에 손가락이 올려진 순간
- 배경에는 흐릿하게 학원 대시보드
- 의미: "복잡한 운영, 결정은 하나"

## 모션 / 애니메이션 방향

| 상황 | 애니메이션 |
|------|-----------|
| 에이전트 실행 중 | Teal 색 pulse ring, 1.5s 주기, 부드러운 opacity 0.4→1→0.4 |
| 승인 완료 | 체크마크 stroke draw-on (0.3s ease-out) + success green flash |
| 이탈 위험 배지 | Amber dot gentle bounce (attention cue, 2s 간격) |
| 대시보드 로딩 | Skeleton UI — off-white → teal light shimmer |
| 히어로 진입 | Fade-up stagger (에이전트 카드들이 100ms 간격으로 아래→위) |
| 에이전트 에러 | Shake (0.3s, 4회) + red border flash |

### 원칙
- 애니메이션은 기능적 의미가 있을 때만 (장식용 금지)
- Duration: 200-400ms (빠른 반응감, 토스 앱 기준)
- Easing: ease-out 기본, spring 효과는 주요 CTA에만
