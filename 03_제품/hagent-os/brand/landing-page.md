### S4. 핵심 기능 1 — 민원 처리 에이전트

레이아웃: 텍스트 좌 / 스크린샷 우 (데스크탑) | 텍스트 상 / 스크린샷 하 (모바일)
배경: off-white #f7f6f1

**태그**: `민원 처리 에이전트`
**헤드라인 (H1)**
> 심야 민원?
> AI가 먼저 분류하고 초안을 씁니다.

**본문**
> 밤 11시 카카오 문자로 환불 요청이 왔습니다. 다음 날 아침, 원장님 화면에는 분류된 민원 유형, 학원법 기준 환불 금액 계산서, 그리고 답변 초안이 기다리고 있습니다. 원장님은 수정하거나, 그대로 승인하면 됩니다.

**기능 리스트 (아이콘 + 텍스트)**
- `MessageCircle` 민원 자동 유형 분류 (환불·불만·문의·칭찬)
- `Scale` 학원법 기준 환불 금액 자동 계산
- `FileText` 답변 초안 생성 (원장 말투 학습)
- `Bell` 카카오 알림톡 또는 문자 자동 발송

**스크린샷**: 승인 카드 UI — 민원 유형 배지 + 내용 요약 + [승인] [수정] 버튼

### S5. 핵심 기능 2 — 이탈 감지 에이전트

레이아웃: 텍스트 우 / 스크린샷 좌 (방향 교차)
배경: white #ffffff

**태그**: `이탈 감지 에이전트`
**헤드라인 (H1)**
> 학생이 떠나기 2주 전,
> AI가 먼저 알아챕니다.

**본문**
> 3주 연속 결석, 질문 감소, 성적 하락 — 이탈 신호는 늘 먼저 옵니다. HagentOS 이탈 감지 에이전트는 출결·성적·상담 데이터를 복합 분석해 위험 학생을 자동으로 분류하고, 맞춤형 상담 문자 초안을 준비합니다.

**기능 리스트**
- `TrendingDown` 이탈 위험 스코어 자동 계산 (HIGH / MED / LOW)
- `CalendarDays` 결석 패턴 3주 연속 감지
- `MessageCircle` 학부모 상담 문자 초안 생성
- `Users` 위험 학생 대시보드 한눈에 보기

**스크린샷**: 이탈 위험 카드 — 학생 이름 + 위험 레벨 배지(amber) + 결석 히스토리 바 차트 + [상담 문자 보내기]

### S6. 핵심 기능 3 — 통합 스케줄러 에이전트

레이아웃: 텍스트 좌 / 스크린샷 우
배경: off-white #f7f6f1

**태그**: `스케줄러 에이전트`
**헤드라인 (H1)**
> 강사 공결?
> 3분 안에 대체 강사가 배정됩니다.

**본문**
> 당일 아침 강사 결근 메시지가 오면, 스케줄러 에이전트가 즉시 가용 강사 리스트를 확인하고 배정 후보를 원장에게 올립니다. 승인 한 번이면 학부모에게 알림이 나가고, 수업 대체 기록이 자동으로 저장됩니다.

**기능 리스트**
- `CalendarDays` 강사·교실·수업 통합 캘린더
- `UserCheck` 가용 강사 자동 매칭
- `Bell` 학부모 대체 강사 안내 발송
- `ReceiptText` 보결 수당 자동 정산

**스크린샷**: 스케줄 캘린더 + 대체 강사 배정 카드 (후보 2명 + 가용 시간 표시)

## 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 14 (App Router) | 정적 생성(SSG), 오픈소스 생태계 |
| 스타일링 | Tailwind CSS + shadcn/ui | Lucide 아이콘 기본 포함 |
| 폰트 | Pretendard (next/font) | 한글 최적화, 무료 |
| 애니메이션 | Framer Motion | 히어로 fade-up, 에이전트 pulse |
| 배포 | Vercel (자동 CD) | GitHub 연동, 무료 티어 |
| GitHub 스타 카운트 | GitHub REST API (ISR, 10분 캐시) | 실시간 배지 |

### 레포 구조 (별도 레포 or 모노레포 /apps/landing)
