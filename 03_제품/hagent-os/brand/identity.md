# Brand Identity

## 브랜드 이름

**HagentOS**
발음: "하젠트 오에스"
의미: **H**agwon(학원 or 학교) + **Agent**(에이전트) + **OS**(운영체제)

약칭으로 "하젠트"도 허용. 영문 표기는 항상 대소문자 그대로 `HagentOS` (붙여 쓰기, O와 S 대문자).

## 브랜드 포지셔닝

HagentOS는 대한민국 9만 4천 개 학원 원장이 매일 3시간을 쏟아붓는 민원·이탈·스케줄 예외 처리를 AI 에이전트 팀에게 위임할 수 있게 하는 **오픈소스 AI 운영체제**다. 기존 학원 관리 SaaS(아이엠 스쿨, 클래스팅 등)가 "기록 도구"라면, HagentOS는 "실행 팀"이다. 원장이 AI를 지시하는 게 아니라, AI 팀이 먼저 상황을 분류하고 초안을 작성한 뒤 사람에게 승인을 요청한다. 데이터는 원장 서버에 남는다. 구독료 없이 GitHub에서 시작한다.

## 브랜드 성격 (Brand Personality)

| 형용사 | 설명 |
|--------|------|
| **신뢰할 수 있는** (Trustworthy) | 원장이 모르는 새 무언가가 처리된 게 아니라, 모든 행동은 승인 후 실행된다. 로그는 언제나 투명하게 열람 가능하다. |
| **실용적인** (Pragmatic) | 화려한 기능 자랑보다 "오늘 아낀 시간 2시간 17분"을 보여준다. 숫자와 현장 언어로 말한다. |
| **한국적인** (Korean-native) | 카카오 알림톡, 학원 법령, 원장-학부모 관계의 미묘한 결을 이해하는 유일한 플랫폼. 글로벌 툴을 억지로 한국어로 번역한 게 아니다. |
| **전문적이지만 친근한** (Professional yet warm) | 토스 앱처럼: 금융(=엄중한 업무)을 다루지만 말투는 편하다. "처리됐습니다"가 아니라 "원장님, 다음 학생 학부모 연락 드렸어요 ✓" |
| **조용히 강한** (Quietly powerful) | 에이전트가 밤새 일하지만 UI는 조용하다. 알림은 꼭 필요할 때만 온다. 힘을 과시하지 않는다. |

## 톤 오브 보이스

### 핵심 원칙
> 학원 원장은 바쁜 전문가다. 앱이 설명을 요구하면 안 된다. 앱이 먼저 일하고, 원장에게는 결정만 남긴다.

### DO — 이렇게 말한다

| 상황 | 예시 문구 |
|------|----------|
| 에이전트 완료 알림 | "박지훈 학부모님 환불 요청, 환급 계산서 초안 완료됐어요. 확인해볼까요?" |
| 이탈 위험 감지 | "이수아 학생, 최근 3주 연속 결석이에요. 상담 문자 초안 드릴게요." |
| 대체 강사 배정 | "김강사 오늘 공결. 박강사 3교시 가능 확인됐어요. 승인하시겠어요?" |
| 오류 안내 | "카카오 연동이 잠깐 끊겼어요. 문자 발송으로 자동 전환했습니다." |
| 온보딩 | "첫 번째 에이전트를 켜볼까요? 5분이면 민원 에이전트가 일 시작합니다." |

### DON'T — 이렇게 말하지 않는다

| 피해야 할 패턴                      | 이유                    |
| ----------------------------- | --------------------- |
| "AI가 귀하의 요청을 처리했습니다."         | 딱딱한 공문체, 거리감          |
| "시스템 오류가 발생했습니다. 코드: ERR_402" | 원장은 개발자가 아님           |
| "스마트 인텔리전스로 최적화된 솔루션"         | 마케팅 버즈워드, 신뢰 손상       |
| "아직 지원하지 않는 기능입니다."           | 막힌 느낌 → "곧 추가될 기능이에요" |
| 느낌표 남발: "완료됐습니다!!!"           | 진지한 업무 맥락에서 유치해 보임    |

### 말투 요약
- 존댓말이되 격식 없이: "~요" 어미 기본
- 학원 현장 용어 그대로: "공결", "보결", "이탈", "원비", "반 편성"
- 숫자는 구체적으로: "3시간" → "2시간 43분"
- 원장을 "원장님"으로 — 앱이 비서처럼 말한다

## 브랜드 스토리 (One Paragraph)

학원 원장은 매일 밤 11시에도 카카오톡을 확인한다. 학부모 민원, 강사 공결 알림, 이번 달 등록 현황, 환불 요청 — 교육을 하고 싶어서 학원을 열었는데 정작 교육에 쏟는 시간은 하루 2시간뿐이다. HagentOS는 이 문제를 푸는 AI 팀이다. 민원 에이전트, 이탈 감지 에이전트, 스케줄 에이전트가 조직처럼 움직이며, 원장의 승인 한 번으로 학원이 돌아간다. 데이터는 원장 서버에 남고, 구독료도 없다. 오픈소스다. 학원 운영에 갇혀있던 원장의 시간을 되돌려주는 것 — 그것이 HagentOS가 존재하는 이유다.
