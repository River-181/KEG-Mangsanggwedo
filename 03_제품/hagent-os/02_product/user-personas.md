## Persona A: 학원 원장 — 김원장 (MVP)

| 항목 | 내용 |
|------|------|
| **이름** | 김원장, 45세 |
| **역할** | 보습학원 원장 10년차 |
| **기관** | 학생 60명, 강사 3명, 서울 강남 |
| **핵심 상황** | 직접 수업도 하면서 민원·상담·이탈 관리까지 혼자 처리 |
| **하루 일과** | 오전: 행정(3시간) → 오후: 수업 → 저녁: 학부모 상담 → 밤: 민원 답장 |
| **가장 아픈 순간** | 밤 11시 학부모 카톡 민원. 강사 갑자기 그만두겠다는 연락 |
| **현재 도구** | 카카오톡 + 클래스업(출결) + 엑셀 + 머릿속 |
| **AI 활용** | ChatGPT 가끔 사용 (민원 답변 참고용) |

### 김원장이 원하는 것

> "내가 수업에 집중할 수 있게 나머지를 누군가 대신 처리해줬으면"

- 아침에 출근하면 "어젯밤 일어난 일" 요약이 있으면 좋겠다
- 민원 답장을 AI가 초안 잡아주면 확인만 하고 보내고 싶다
- 이번 달 누가 그만둘 것 같은지 미리 알고 싶다
- 강사 스케줄 조정을 대신 해줬으면 좋겠다

### 김원장이 싫어하는 것

- 설정이 복잡한 도구 (또 하나의 관리 대상이 됨)
- AI가 알아서 보내버리는 것 (내가 확인은 해야 한다)
- 기능이 너무 많아서 뭘 써야 할지 모르는 것

| 지표 | 수치 |
|------|------|
| 전국 학원 | 94,485개 ([경향신문](https://www.khan.co.kr/article/202510131343001)) |
| 보습/입시/IT 학원 | ~50,000개 (추정, 학과 교습 53%) |
| 10~100명 규모 | ~35,000개 (추정) |
| 원장+실장 | ~60,000~80,000명 |
