# HagentOS MVP Quick Start

## Requirements

- Node.js 18 or newer
- macOS, Windows, or Linux

## Run

1. Open a terminal in this folder.
2. Run:

```bash
npm start
```

3. Open this URL in a browser:

```text
http://127.0.0.1:4173
```

## Shortcut Launchers

- macOS: double-click `run.command`
- Windows: double-click `start.bat`

## Notes

- This MVP uses only built-in Node.js modules.
- No extra package install is required.
- Data is stored in memory for demo purposes.
