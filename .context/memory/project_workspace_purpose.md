이 워크스페이스는 KEG 바이브코딩 콘테스트를 위한 AI native 협업 공간.

**Why:** 2명의 인간 팀원 + AI 에이전트들이 대회 준비→개발→제출→회고까지 유기적으로 협업하는 공간. 단순 코드 저장소가 아니라 지식·전략·프로세스를 공유하는 허브.

**How to apply:**
- 인터페이스: 옵시디언 (사용자가 문서 탐색·편집에 활용)
- 프로젝트 관리: 깃허브 (추후 연동 예정)
- app/ 디렉토리: 실제 제출 작품 개발 공간 (예정)
- 기초 자료: 대회 분석, 전략 플레이북, 홍보자료 등 이미 수집됨
- 각 카테고리는 탑다운 색인(맵) 아래에서 접근 가능하도록 구조화 필요
- `vibe_contest_master_playbook_v0_1.md`: GPT pro가 정리한 전략 플레이북 (참고용, 필수 아님)
