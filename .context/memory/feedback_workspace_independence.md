이 워크스페이스는 사용자의 로컬 환경과 연결되지 않은 독립적인 공간으로 정의해야 한다.

**Why:** 팀원 2명 + 여러 AI 에이전트가 동일한 맥락에서 작업해야 하므로, 개인 로컬 설정에 의존하면 안 된다. 모든 설정·스킬·MCP·메모리가 이 프로젝트 디렉토리 안에서 자기완결적이어야 한다.

**How to apply:** CLAUDE.md, AGENTS.md, 에이전트 정의, 스킬, MCP 설정 등을 프로젝트 루트에 배치. 글로벌 설정 참조 대신 프로젝트 내 설정을 우선.
