## 핸드오프 계약

모든 에이전트 출력은 최소한 아래 6개 블록을 포함한다:

```
[Task] — 수행한 작업
[Assumptions] — 전제한 가정
[Inputs used] — 참조한 자료
[Output] — 산출물
[Open risks] — 미해결 위험
[Next recommended action] — 다음 추천 행동
```
