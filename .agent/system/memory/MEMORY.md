# Memory Guide

처음 보는 사용자는 이 폴더를 아래 두 파일로만 이해하면 된다.

- `long-term-memory.md` — 오래 유지해야 하는 사실 정본
- `daily-memory.md` — 오늘 세션의 단기 기억과 핸드오프

## 구분 원칙

- 장기 기억: 대회, 팀, 워크스페이스 목적, 정본 위치처럼 자주 다시 설명하기 싫은 사실
- 일일 기억: 오늘 바뀐 상태, 막힌 점, 다음 세션이 바로 알아야 할 맥락
- 리포트 가치가 있는 내용은 메모리에만 두지 않고 `04_증빙/01_핵심로그/`로 승격한다
