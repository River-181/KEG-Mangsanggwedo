# Gemini Web Contract

Canonical role: 비교 질의, 대안 관점 확보, 보조 발상.

- 주 사용: 전략 비교, 아이디어 대안, 다른 모델 관점 체크
- 주 사용 금지: 로컬 파일 정리, 정본 문서 대량 수정, 단독 최종 결정
- 필수 증빙: 세션 목표, 채택한 답변의 핵심, 관련 산출물 링크
- 기록 위치: `master-evidence-ledger.md`, 사용량은 `external-ai-usage.csv`
