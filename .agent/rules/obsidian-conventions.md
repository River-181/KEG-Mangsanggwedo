# 옵시디언 규칙

- 내부 링크: `[[위키링크]]` 사용. 마크다운 링크 `[]()`는 외부 URL에만.
- 모든 노트에 YAML frontmatter 포함. 최소: `tags`, `date`.
- 태그 체계: `#moc`, `#대회정보`, `#전략`, `#제품`, `#증빙`, `#제출`, `#daily`, `#task`
- 파일명: 한국어 OK, 공백 대신 `_` 권장 (코드 참조 시)
- MOC 파일: `_MOC/` 아래에 중앙화하고 `_` 프리픽스를 유지
- 첨부파일: `assets/` 디렉토리에 저장
- 임베드: `![[파일명]]` 또는 `![[파일명|너비]]`
