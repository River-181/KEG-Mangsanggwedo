# Excalidraw MCP

## 역할

- 워크스페이스 아키텍처 시각화
- AI 리포트용 시스템 다이어그램 제작
- 프로세스, 데이터 흐름, 시퀀스 다이어그램 정리

## 연결 원칙

- Excalidraw MCP 연결 정보는 팀원이 각자 설정한다.
- 그러나 사용 목적, 다이어그램 산출 위치, 운영 규칙은 이 저장소가 정본이다.

## 산출물 위치

- 발표/설명용 다이어그램: `assets/demo/` 또는 `05_제출/`
- 설계 정본 note: `04_증빙/02_분석자료/` 또는 `02_전략/`

## 팀 적용 체크

1. Claude Code 또는 해당 에이전트 런타임에서 Excalidraw MCP 연결
2. [[workspace-atlas]] 기준 다이어그램 노드명 통일
3. 결과 note는 `[[master-evidence-ledger]]`에 기록
